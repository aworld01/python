Create a VirtualEnvironment for python3

Open Command prompt in your desired folder and fire these commands.

  1. pip install virtualenv: to install VirtualEnvironment package in your pc if not installed.
  2. virtualenv VirtualEnv: to create a fresh VirtualEnvronment.
    Note: You have to install packages in fresh created VirtualEnvironment by pip install 'package name'.
    or
    virtualenv --system-site-packages VirtualEnv: to create a virtual_Environment with all installed packages.
  3. VirtualEnv\Scripts\activate: to activate VirtualEnvironment.
    source VirtualEnv/Scripts/activate: if you are using Linux os.
  4. pip list: to check list of installed packages.
  5. deactivate: to deactivate VirtualEnvironment.

Requirements.txt for python3

  1. pip freeze > requirements.txt: to create a requirements.txt file.
  2. pip install -r requirements.txt: to install all packages that is listed in requirements.txt file.
